## Задача 2.3 - Числа Падована

### Постановка задачи
Организовать и распечатать последовательность чисел Падована, не превосходящих число m, введенное с клавиатуры.

Числа Падована представлены следующим рядом:
**1, 1, 1, 2, 2, 3, 4, 5, 7, 9, 12, 16, 21, 28, 37, 49, 65, 86, 114, 151, 200, 265, ...**

Формулы для вычисления:
- **P(0) = P(1) = P(2) = 1**
- **P(n) = P(n-2) + P(n-3)** для n ≥ 3

### Математическая модель

**Рекуррентная формула:**
P(n) = P(n-2) + P(n-3)

**Начальные условия:**
- P(0) = 1
- P(1) = 1
- P(2) = 1

**Последовательность вычисляется так:**
- P(3) = P(1) + P(0) = 1 + 1 = 2
- P(4) = P(2) + P(1) = 1 + 1 = 2
- P(5) = P(3) + P(2) = 2 + 1 = 3
- P(6) = P(4) + P(3) = 2 + 2 = 4
- и так далее...

### Список идентификаторов

| Имя переменной | Тип данных | Описание |
|----------------|------------|----------|
| m | int | Верхняя граница (числа не больше m) |
| p0, p1, p2 | int | Три последних числа последовательности |
| p_next | int | Следующее число последовательности |

### Код программы

```c
#include <stdio.h>

int main() {
    int m;
    
    // Ввод верхней границы
    printf("Введите число m: ");
    scanf("%d", &m);
    
    // Проверка корректности ввода
    if (m < 1) {
        printf("Ошибка: m должно быть больше 0\n");
        return 1;
    }
    
    printf("\nЧисла Падована, не превышающие %d:\n", m);
    printf("============================================\n");
    
    // Первые три числа (все равны 1)
    int p0 = 1;  // P(n-3)
    int p1 = 1;  // P(n-2)
    int p2 = 1;  // P(n-1)
    
    // Печатаем первые три числа, если они не превышают m
    printf("%d ", p0);
    if (m >= 1) {
        printf("%d ", p1);
    }
    if (m >= 1) {
        printf("%d ", p2);
    }
    
    // Вычисляем остальные числа
    while (1) {
        int p_next = p0 + p1;  // P(n) = P(n-2) + P(n-3)
        
        if (p_next > m) {
            break;  // если следующее число больше m - выходим
        }
        
        printf("%d ", p_next);
        
        // Сдвигаем значения для следующей итерации
        p0 = p1;
        p1 = p2;
        p2 = p_next;
    }
    
    printf("\n============================================\n");
    
    return 0;
}
