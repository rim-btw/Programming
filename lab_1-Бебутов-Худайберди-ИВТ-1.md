## Задача 1.3 - Вычисление выражения u(x,y)

### Постановка задачи
Вычислить значение выражения:

**u(x, y) = (1 + sin²(x + y)) / (2 + |x - 2x²/(1 + |sin(x + y)|)|)**

где x и y вводятся с клавиатуры.

### Математическая модель

Пусть:
- A = sin(x + y)
- B = |A| — модуль синуса
- C = 2x²/(1 + B)
- D = |x - C|

Тогда:
**u = (1 + A²) / (2 + D)**

где:
- x, y — аргументы (действительные числа)
- u — результат вычисления

### Список идентификаторов

| Имя переменной | Тип данных | Описание                                      |
|----------------|------------|-----------------------------------------------|
| x              | double     | Аргумент x                                    |
| y              | double     | Аргумент y                                    |
| sin_xy         | double     | sin(x + y)                                    |
| numerator      | double     | Числитель: 1 + sin²(x + y)                    |
| denominator    | double     | Знаменатель: 2 + |x - 2x²/(1 + |sin(x + y)|)| |
| u              | double     | Результат вычисления                          |

### Код программы

```c
#include <stdio.h>
#include <math.h>

int main() {
    double x, y, u;
    
    // Ввод данных
    printf("Enter x: ");
    scanf("%lf", &x);
    
    printf("Enter y: ");
    scanf("%lf", &y);
    
    // Вычисление sin(x + y)
    double sin_xy = sin(x + y);
    
    // Числитель
    double numerator = 1 + pow(sin_xy, 2);
    
    // Знаменатель
    double denominator = 2 + fabs(x - (2 * pow(x, 2)) / (1 + fabs(sin_xy)));
    
    // Результат
    u = numerator / denominator;
    
    // Вывод результата
    printf("u(x, y) = %lf\n", u);
    
    return 0;
}

## Задача 1.4 - Вычисление выражения h(x)

### Постановка задачи
Вычислить значение выражения:

**h(x) = - (x - a) / ∛(x² + a²) - (4 · ⁴√((x² + b²)³)) / (2 + a + b + ∛((x - c)²))**

Для трех наборов значений:

| Набор    | a    | b   | c   | x      |
|----------|------|-----|-----|--------|
| 1        | 0.12 | 3.5 | 2.4 | 1.4    |
| 2        | 0.12 | 3.5 | 2.3 | 1.6    |
| 3        | 0.27 | 3.9 | 2.8 | 1.8    |

### Математическая модель

Выражение состоит из двух частей:

**Первая часть:** -(x - a) / ∛(x² + a²)

**Вторая часть:** (4 · ⁴√((x² + b²)³)) / (2 + a + b + ∛((x - c)²))

**Результат:** h(x) = Часть1 - Часть2

где:
- ∛ — кубический корень (cbrt)
- ⁴√ — корень четвертой степени (степень 1/4)

### Список идентификаторов

| Имя переменной | Тип данных | Описание             |
|----------------|------------|----------------------|
| a, b, c        | double     | Параметры выражения  |
| x              | double     | Аргумент функции     |
| h              | double     | Результат вычисления |

### Код программы

```c
#include <stdio.h>
#include <math.h>

int main() {
    double a, b, c, x, h;
    
    // ===== Набор 1 =====
    a = 0.12; b = 3.5; c = 2.4; x = 1.4;
    h = -(x - a) / cbrt(pow(x, 2) + pow(a, 2)) 
        - (4 * pow(pow(x, 2) + pow(b, 2), 3.0/4.0)) 
          / (2 + a + b + cbrt(pow(x - c, 2)));
    printf("Case 1 (a=0.12, b=3.5, c=2.4, x=1.4): h = %.6lf\n", h);
    
    // ===== Набор 2 =====
    a = 0.12; b = 3.5; c = 2.3; x = 1.6;
    h = -(x - a) / cbrt(pow(x, 2) + pow(a, 2)) 
        - (4 * pow(pow(x, 2) + pow(b, 2), 3.0/4.0)) 
          / (2 + a + b + cbrt(pow(x - c, 2)));
    printf("Case 2 (a=0.12, b=3.5, c=2.3, x=1.6): h = %.6lf\n", h);
    
    // ===== Набор 3 =====
    a = 0.27; b = 3.9; c = 2.8; x = 1.8;
    h = -(x - a) / cbrt(pow(x, 2) + pow(a, 2)) 
        - (4 * pow(pow(x, 2) + pow(b, 2), 3.0/4.0)) 
          / (2 + a + b + cbrt(pow(x - c, 2)));
    printf("Case 3 (a=0.27, b=3.9, c=2.8, x=1.8): h = %.6lf\n", h);
    
    return 0;
}

## Задача 2.1 - Координаты планеты Марс относительно Земли

### Постановка задачи
Вычислить используя цикл for координаты планеты Марс относительно Земли с течением времени t. Распечатать на экране координаты для каждой итерации по t.

Координаты задаются формулами:

**x = r₁·cos(w₁·t) - r₂·cos(w₂·t)**

**y = r₁·sin(w₁·t) - r₂·sin(w₂·t)**

**w₁ = 2π/T₁**

**w₂ = 2π/T₂**

где:
- r₁ — радиус орбиты Марса
- r₂ — радиус орбиты Земли
- T₁ — период обращения Марса
- T₂ — период обращения Земли
- t — время

### Математическая модель

| Параметр   | Значение | Единица измерения |
|------------|----------|-------------------|
| r₁ (Марс)  | 227.9    | млн км            |
| r₂ (Земля) | 149.6    | млн км            |
| T₁ (Марс)  | 687      | дни               |
| T₂ (Земля) | 365      | дни               |

**Угловые скорости:**
- w₁ = 2 · 3.14159 / 687 ≈ 0.00915 рад/день
- w₂ = 2 · 3.14159 / 365 ≈ 0.01721 рад/день

**Координаты в момент времени t:**
- x(t) = r₁·cos(w₁·t) - r₂·cos(w₂·t)
- y(t) = r₁·sin(w₁·t) - r₂·sin(w₂·t)

### Список идентификаторов

| Имя переменной | Тип данных |  Описание                           |
|----------------|------------|-------------------------------------|
| r1, r2         | double     | Радиусы орбит Марса и Земли         |
| T1, T2         | double     | Периоды обращения Марса и Земли     |
| w1, w2         | double     | Угловые скорости                    |
| t              | double     | Время (дни)                         |
| x, y           | double     | Координаты Марса относительно Земли |

### Код программы

```c
#include <stdio.h>
#include <math.h>

int main() {
    // Радиусы орбит (млн км)
    double r1 = 227.9;   // Марс
    double r2 = 149.6;   // Земля
    
    // Периоды обращения (дни)
    double T1 = 687.0;   // Марс
    double T2 = 365.0;   // Земля
    
    // Угловые скорости
    double w1 = 2 * M_PI / T1;
    double w2 = 2 * M_PI / T2;
    
    double t, x, y;
    
    // Заголовок таблицы
    printf("==============================================\n");
    printf("  t (дни)     x (млн км)     y (млн км)\n");
    printf("==============================================\n");
    
    // Цикл по времени от 0 до 800 дней с шагом 100
    for (t = 0; t <= 800; t += 100) {
        x = r1 * cos(w1 * t) - r2 * cos(w2 * t);
        y = r1 * sin(w1 * t) - r2 * sin(w2 * t);
        printf("  %.0f          %.2f          %.2f\n", t, x, y);
    }
    
    printf("==============================================\n");
    
    return 0;
}

## Задача 2.2 - Вычисление определенного интеграла методом трапеций

### Постановка задачи
Вычислить определенный интеграл от заданной функции методом трапеций:

**∫ₐᵇ f(x) dx = ∫₀¹ e^(x+2) dx**

Функция: **f(x) = e^(x+2)**

Результат интегрирования сравнить с вычисленным вручную (аналитически) и убедиться в корректности результата.

### Математическая модель

**Метод трапеций:**
∫ₐᵇ f(x) dx ≈ h · [ (f(a) + f(b))/2 + Σᵢ₌₁ⁿ⁻¹ f(a + i·h) ]

где:
- h = (b - a)/n — шаг интегрирования
- n — количество трапеций (чем больше, тем точнее)
- a = 0 — нижний предел
- b = 1 — верхний предел

**Аналитическое вычисление:**
∫₀¹ e^(x+2) dx = e^(x+2) |₀¹ = e^(1+2) - e^(0+2) = e³ - e²

**Точное значение:**
e³ - e² ≈ 20.0855 - 7.3891 = 12.6964

### Список идентификаторов

| Имя переменной | Тип данных | Описание                                      |
|----------------|------------|-----------------------------------------------|
| a, b           | double     | Нижний и верхний пределы интегрирования       |
| n              | int        | Количество трапеций                           |
| h              | double     | Шаг интегрирования                            |
| sum            | double     | Сумма значений функции в промежуточных точках |
| integral       | double     | Приближенное значение интеграла               |
| exact          | double     | Точное значение интеграла                     |

### Код программы

```c
#include <stdio.h>
#include <math.h>

int main() {
    double a = 0.0;   // нижний предел
    double b = 1.0;   // верхний предел
    int n = 1000;     // количество трапеций (чем больше, тем точнее)
    double h = (b - a) / n;  // шаг
    double sum = 0.0;
    double x;
    
    // Функция f(x) = e^(x+2)
    // Вычисляем сумму значений в промежуточных точках
    for (int i = 1; i < n; i++) {
        x = a + i * h;
        sum += exp(x + 2);
    }
    
    // Формула трапеций
    double integral = h * ((exp(a + 2) + exp(b + 2)) / 2 + sum);
    
    // Точное значение (аналитически)
    double exact = exp(b + 2) - exp(a + 2);
    
    // Вывод результатов
    printf("============================================\n");
    printf("ВЫЧИСЛЕНИЕ ОПРЕДЕЛЕННОГО ИНТЕГРАЛА\n");
    printf("============================================\n");
    printf("Функция: f(x) = e^(x+2)\n");
    printf("Пределы: от %.1f до %.1f\n", a, b);
    printf("Количество трапеций: %d\n", n);
    printf("============================================\n");
    printf("Метод трапеций: %.6lf\n", integral);
    printf("Точное значение: %.6lf\n", exact);
    printf("============================================\n");
    printf("Погрешность: %.10lf\n", fabs(integral - exact));
    printf("============================================\n");
    
    return 0;
}

## Задача 2.3 - Числа Падована

### Постановка задачи
Организовать и распечатать последовательность чисел Падована, не превосходящих число m, введенное с клавиатуры.

Числа Падована представлены следующим рядом:
**1, 1, 1, 2, 2, 3, 4, 5, 7, 9, 12, 16, 21, 28, 37, 49, 65, 86, 114, 151, 200, 265, ...**

Формулы для вычисления:
- **P(0) = P(1) = P(2) = 1**
- **P(n) = P(n-2) + P(n-3)** для n ≥ 3

### Математическая модель

**Рекуррентная формула:**
P(n) = P(n-2) + P(n-3)

**Начальные условия:**
- P(0) = 1
- P(1) = 1
- P(2) = 1

**Последовательность вычисляется так:**
- P(3) = P(1) + P(0) = 1 + 1 = 2
- P(4) = P(2) + P(1) = 1 + 1 = 2
- P(5) = P(3) + P(2) = 2 + 1 = 3
- P(6) = P(4) + P(3) = 2 + 2 = 4
- и так далее...

### Список идентификаторов

| Имя переменной | Тип данных | Описание                               |
|----------------|------------|----------------------------------------|
| m              | int        | Верхняя граница (числа не больше m)    |
| p0, p1, p2     | int        | Три последних числа последовательности |
| p_next         | int        | Следующее число последовательности     |

### Код программы

```c
#include <stdio.h>

int main() {
    int m;
    
    // Ввод верхней границы
    printf("Введите число m: ");
    scanf("%d", &m);
    
    // Проверка корректности ввода
    if (m < 1) {
        printf("Ошибка: m должно быть больше 0\n");
        return 1;
    }
    
    printf("\nЧисла Падована, не превышающие %d:\n", m);
    printf("============================================\n");
    
    // Первые три числа (все равны 1)
    int p0 = 1;  // P(n-3)
    int p1 = 1;  // P(n-2)
    int p2 = 1;  // P(n-1)
    
    // Печатаем первые три числа, если они не превышают m
    printf("%d ", p0);
    if (m >= 1) {
        printf("%d ", p1);
    }
    if (m >= 1) {
        printf("%d ", p2);
    }
    
    // Вычисляем остальные числа
    while (1) {
        int p_next = p0 + p1;  // P(n) = P(n-2) + P(n-3)
        
        if (p_next > m) {
            break;  // если следующее число больше m - выходим
        }
        
        printf("%d ", p_next);
        
        // Сдвигаем значения для следующей итерации
        p0 = p1;
        p1 = p2;
        p2 = p_next;
    }
    
    printf("\n============================================\n");
    
    return 0;
}

## Задача 2.4 - Сумма цифр трехзначного числа

### Постановка задачи
С клавиатуры вводится трехзначное число, считается сумма его цифр. 
Если сумма цифр числа больше 10, то вводится следующее трехзначное число.
Если сумма меньше либо равна 10 — программа завершается.

### Математическая модель

**Для трехзначного числа ABC (где A - сотни, B - десятки, C - единицы):**

- A = число / 100 (целочисленное деление)
- B = (число / 10) % 10 (десятки)
- C = число % 10 (единицы)

**Сумма цифр: S = A + B + C**

**Условие продолжения:** S > 10 → вводим новое число
**Условие остановки:** S ≤ 10 → программа завершается

### Список идентификаторов

| Имя переменной | Тип данных | Описание                    |
|----------------|------------|-----------------------------|
| number         | int        | Введенное трехзначное число |
| sum            | int        | Сумма цифр числа            |
| hundreds       | int        | Цифра сотен                 |
| tens           | int        | Цифра десятков              |
| units          | int        | Цифра единиц                |

### Код программы

```c
#include <stdio.h>

int main() {
    int number;
    int sum;
    int hundreds, tens, units;
    
    printf("============================================\n");
    printf("ПРОГРАММА: СУММА ЦИФР ТРЕХЗНАЧНОГО ЧИСЛА\n");
    printf("============================================\n");
    printf("Правила:\n");
    printf("- Если сумма цифр > 10 - вводим следующее число\n");
    printf("- Если сумма цифр <= 10 - программа завершается\n");
    printf("============================================\n\n");
    
    while (1) {  // Бесконечный цикл
        // Ввод числа
        printf("Введите трехзначное число: ");
        scanf("%d", &number);
        
        // Проверка, что число трехзначное
        if (number < 100 || number > 999) {
            printf("ОШИБКА! Число должно быть трехзначным (от 100 до 999).\n");
            printf("Попробуйте снова.\n\n");
            continue;  // Просим ввести число заново
        }
        
        // Вычисляем цифры
        hundreds = number / 100;           // сотни
        tens = (number / 10) % 10;         // десятки
        units = number % 10;               // единицы
        
        // Вычисляем сумму цифр
        sum = hundreds + tens + units;
        
        // Вывод результата
        printf("Число: %d\n", number);
        printf("Цифры: %d + %d + %d = %d\n", hundreds, tens, units, sum);
        
        // Проверка условия
        if (sum <= 10) {
            printf("\n============================================\n");
            printf("Сумма цифр = %d <= 10\n", sum);
            printf("ПРОГРАММА ЗАВЕРШЕНА!\n");
            printf("============================================\n");
            break;  // Выход из цикла
        } else {
            printf("Сумма цифр = %d > 10\n", sum);
            printf("Введите следующее число...\n\n");
            // Цикл продолжается
        }
    }
    
    return 0;
}
