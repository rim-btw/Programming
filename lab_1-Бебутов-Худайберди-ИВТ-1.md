## Задача 2.2 - Вычисление определенного интеграла методом трапеций

### Постановка задачи
Вычислить определенный интеграл от заданной функции методом трапеций:

**∫ₐᵇ f(x) dx = ∫₀¹ e^(x+2) dx**

Функция: **f(x) = e^(x+2)**

Результат интегрирования сравнить с вычисленным вручную (аналитически) и убедиться в корректности результата.

### Математическая модель

**Метод трапеций:**
∫ₐᵇ f(x) dx ≈ h · [ (f(a) + f(b))/2 + Σᵢ₌₁ⁿ⁻¹ f(a + i·h) ]

где:
- h = (b - a)/n — шаг интегрирования
- n — количество трапеций (чем больше, тем точнее)
- a = 0 — нижний предел
- b = 1 — верхний предел

**Аналитическое вычисление:**
∫₀¹ e^(x+2) dx = e^(x+2) |₀¹ = e^(1+2) - e^(0+2) = e³ - e²

**Точное значение:**
e³ - e² ≈ 20.0855 - 7.3891 = 12.6964

### Список идентификаторов

| Имя переменной | Тип данных | Описание |
|----------------|------------|----------|
| a, b | double | Нижний и верхний пределы интегрирования |
| n | int | Количество трапеций |
| h | double | Шаг интегрирования |
| sum | double | Сумма значений функции в промежуточных точках |
| integral | double | Приближенное значение интеграла |
| exact | double | Точное значение интеграла |

### Код программы

```c
#include <stdio.h>
#include <math.h>

int main() {
    double a = 0.0;   // нижний предел
    double b = 1.0;   // верхний предел
    int n = 1000;     // количество трапеций (чем больше, тем точнее)
    double h = (b - a) / n;  // шаг
    double sum = 0.0;
    double x;
    
    // Функция f(x) = e^(x+2)
    // Вычисляем сумму значений в промежуточных точках
    for (int i = 1; i < n; i++) {
        x = a + i * h;
        sum += exp(x + 2);
    }
    
    // Формула трапеций
    double integral = h * ((exp(a + 2) + exp(b + 2)) / 2 + sum);
    
    // Точное значение (аналитически)
    double exact = exp(b + 2) - exp(a + 2);
    
    // Вывод результатов
    printf("============================================\n");
    printf("ВЫЧИСЛЕНИЕ ОПРЕДЕЛЕННОГО ИНТЕГРАЛА\n");
    printf("============================================\n");
    printf("Функция: f(x) = e^(x+2)\n");
    printf("Пределы: от %.1f до %.1f\n", a, b);
    printf("Количество трапеций: %d\n", n);
    printf("============================================\n");
    printf("Метод трапеций: %.6lf\n", integral);
    printf("Точное значение: %.6lf\n", exact);
    printf("============================================\n");
    printf("Погрешность: %.10lf\n", fabs(integral - exact));
    printf("============================================\n");
    
    return 0;
}
